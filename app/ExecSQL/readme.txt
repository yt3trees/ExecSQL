# README
ExecSQLは.sqlファイルをより少ないプロセスで実行することができるアプリケーションです。

.sqlファイルを複数もしくは1つ選択した上で右クリックしてください。
「送る」メニューに表示されているExecSQLを選択することでアプリケーションが起動されます。

接続先データベースを入力し、「RUN」ボタンを押せばそのまま実行できます。
実行した.sqlファイルのクエリがSELECTの場合はその結果を表示するボタン、
UPDATEやCREATEの場合はその成否が表示されます。

※現時点では同一.sqlファイル内での複数のSELECTには対応していません。

プログラムについての問い合わせは下記githubのリポジトリへISSUEお願いします。
